# 1971 - Claude Notes

## Project Overview
Static data visualization exploring the economic shift when the US abandoned the gold standard in 1971. HTML/CSS/JS + Chart.js. Deployed to GitHub Pages.

## Stack
- Static HTML/CSS/JS (no build step)
- Chart.js for visualizations
- GitHub Pages deployment

## Development
```bash
open index.html   # Local preview
git push origin main  # Deploys to https://nulljosh.github.io/1971
```

## Data Sources
- Federal Reserve Economic Data (FRED)
- Bureau of Labor Statistics
- "A Generation of Sociopaths" by Bruce Cannon Gibney

## Key Files
- `index.html` - Main visualization page
- `preview.png` - Social preview image

## Status
Live at https://nulljosh.github.io/1971. Idea-stage expansions: timeline slider, country comparisons, inflation calculator.
